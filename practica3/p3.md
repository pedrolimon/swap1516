##configurar una máquina e instalarle el nginx como balanceador de carga##
Lo primero que hay que realizar es preparar la máquina para poder instalar  
'nginx', importando la clave del repositorio de software mediante:  
~~~~
	'cd /tmp/'  
~~~~
~~~~
    'wget http://nginx.org/keys/nginx_signing.key'  
~~~~
~~~~
    'apt-key add /tmp/nginx_signing.key'  
~~~~
~~~~
    'rm -f /tmp/nginx_signing.key'  
~~~~
Acto seguido añadimos el repositorio al fichero /etc/apt/sources.list:  
~~~~
    'echo "deb http://nginx.org/packages/ubuntu/ lucid nginx" >> /etc/apt/sources.list'  
~~~~
~~~~
    'echo "deb-src http://nginx.org/packages/ubuntu/ lucid nginx" >> /etc/apt/sources.list'  
~~~~
Una vez realizados estos pasos previos podemos instalar el paquete 'nginx':  
~~~~
	'apt-get update'  
~~~~
~~~~
    'apt-get install nginx'  
~~~~
El siguiente paso que debemos realizar una vez instalado 'nginx' sería la de consigurarlo,  
para ello tenemos que modificar el archivo '/etc/nginx/conf.d/default.conf':  
    * definir la máquinas que formaran el cluster web indicando las IP de estas máquinas  
    * modificar la sección 'server' con los datos necesarios
